Random Quote Generator

 Астанин Дмитрий Сергеевич

## Описание

Приложение «Random Quote Generator» позволяет генерировать случайные цитаты из предопределённого списка. Программа отображает текущую цитату с указанием автора и темы, а также ведёт историю сгенерированных цитат.

### Функциональность

- Генерация случайной цитаты по нажатию кнопки.
- Отображение истории последних 20 сгенерированных цитат.
- Фильтрация цитат по автору и теме.
- Сохранение и загрузка истории и списка цитат из JSON‑файла.
- Проверка корректности ввода (пустые строки при добавлении новых цитат).
